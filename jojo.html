JoJo HTML Game

Medium: Web Game
Timeline: December 2016
Role: Creator & Developer
Synopsis: Inspired by the anime JoJo's Bizzare Adventure, I adapted an arc of its story into a roleplaying game created with HTML, CSS and JavaScript. With beta-tested difficulty and exciting action, behold the great tale of Jotaro defeating his arch-nemesis Dio in Egypt.

This game has been heavily reduced to be able to be hosted online. Original game consists of videos instead of the static images you see in the current game. Contact me and I will send you a .zip file of the original.
